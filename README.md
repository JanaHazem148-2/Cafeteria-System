# Cafeteria System

A Java desktop application for managing a cafeteria — handling students, menu items, and orders through a graphical interface built with JavaFX.

---

## Features

- **Student Management** — register and manage student accounts
- **Menu Item Management** — add, update, and remove cafeteria items
- **Order Processing** — place and track orders per student
- **Persistent Storage** — data is saved locally using Java object serialization (`.dat` files)
- **MySQL Integration** — optional database backend via MySQL Connector/J

---

## Prerequisites

- **Java JDK** (version compatible with your JavaFX version)
- **JavaFX SDK** — add as a library in your IDE
- **MySQL Server** (optional, if using database backend)
- **MySQL Connector/J 9.4.0** — add the JAR as a project library
- **IntelliJ IDEA** (recommended)

---

## Setup & Running

### 1. Clone the repository

```bash
git clone <repository-url>
cd Cafeteria_System
```

### 2. Open in IntelliJ IDEA

- Open the project folder in IntelliJ IDEA.
- The `.iml` module file will be detected automatically.

### 3. Configure libraries

Add the following libraries under **File → Project Structure → Libraries**:

- `javafx.controls` — your local JavaFX SDK
- `mysql-connector-j-9.4.0.jar` — MySQL JDBC driver
- Any additional `lib/` or `bin/` dependencies

### 4. Configure VM options (JavaFX)

In your Run Configuration, add VM options pointing to your JavaFX SDK:

```
--module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml
```

### 5. (Optional) Set up MySQL

If the application uses a MySQL database, create the required database and update the connection settings (host, port, database name, username, password) in the source configuration file.

### 6. Run

Execute the main application class from IntelliJ IDEA or via the command line.

| File | Contents |
|------|----------|

| `items.dat` | Cafeteria menu items |
| `orders.dat` | Order history |

> **Note:** These files are auto-generated by the application on first run. Do not edit them manually.
